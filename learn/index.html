Ecosystem

Ember's ecosystem of libraries solve the hard problems around building, client to server communication and server side rendering.

You build your application using Ember CLI. If you're just getting started with Ember, or are unfamiliar with Ember CLI, head over to the Ember CLI User Guides. The default build process might not suit you perfectly, so you need to extend Ember CLI itself.

While it's not a requirement, Ember includes a complimentary library for client to server communication called Ember Data.

If you want your application to be rendered on the server, FastBoot is the solution for your Ember application.
