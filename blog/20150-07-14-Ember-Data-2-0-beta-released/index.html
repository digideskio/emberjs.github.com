We are pleased to announce the release of Ember Data 2.0.beta1.

Ember Data 2.0 beta is the first release in the 2.x series. This means that many features deprecated during the 1.x series will be removed in Ember Data 2.0. If you encounter any unexpected changes in features not marked as deprecated in Ember Data 1.13 while testing Ember Data 2.0 beta, please report them immediately. We would like to fix these unintentional regressions before the final release of Ember Data 2.0.

Breaking Change

As we explained in the The Ember 2.x Project blog post, going forward Ember Data will be syncing up its releases and version numbers with Ember.js releases.
